Almost all West Coast Swing moves are a variation of these basic patterns. You should think of these patterns as 2-count-long elements, which are the essential ingredients sandwiched between the '1&2' and the anchor. The example links given below aren't necessarily the patterns themselves, but just the elements that make it that pattern. The 'Texas Tommy' example below is a Texas Tommy with an unwinding spin into a wrap into a go-around, but the element that makes it a whip is the Texas Tommy.
